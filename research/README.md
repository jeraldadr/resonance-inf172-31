
 # Landing page for research

 This is a code bundle for Landing page for research. The original project is available at https://www.figma.com/design/j05WGltlDo4UjuGvwRuclz/Landing-page-for-research.

 **Deployed research page:** https://resonance-inf172-31-research.vercel.app/

 ---

 ## Additional bundles in this folder

 - **172 Wireframes** (`./wireframes`): code bundle for the 172 wireframes. The original project is available at https://www.figma.com/design/UBjTYzb6uEIWxk2tmS2QVw/172-Wireframes.

   - To run the wireframes locally: from the `research/wireframes` directory, run `npm i` and then `npm run dev`.

 ## Running the main research landing page

 From the `research` directory:

 - Run `npm i` to install the dependencies.
 - Run `npm run dev` to start the development server.
